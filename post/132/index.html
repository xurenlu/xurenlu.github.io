Ansys 计算估计相关内容.

2005-05-06

A.估计求解时间:
ansys可以根据计算机的性能估计求解是加尼，一般步骤如下:
1.使用rspeed命令获得计算机性能:
命令:rspeed,mips,smflop,vmflop
GUI: Main Menu->Run Time Stats-> System Settings

2:使用RITER命令得到分析的迭带次数:
命令:RITER,NITER
GUI: Main Menu->Run-Time Stats->Iter Settings

3.获得求解时间估计值:
命令: RTIMST
GUI: Main Menu-> Run-Time Stats-> Individual Stats

B 估计生成文件大小:
使用RFilSZ可以对文件大小进行估计，包括的文件类型有:.esav,.emat,.erot,.tri,.full,.rst,.rth,.rmg,.rfl;

C.估计内存要求:
命令:RWFRNT;
D 给出模型单元信息
命令:RSTAT
E 给出内存统计量
命令:RMEMRY
GUI:main menu->run-time stats ->individual stats
F 给出模型所有统计信息:
命令:RALL;
GUI: Main Menu-> Run-Time Stats-> All Statistics.
