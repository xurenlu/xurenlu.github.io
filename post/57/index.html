昨天想到一个制造垃圾信息推广自己网站的方法

2006-01-09

昨天想到一个制造垃圾信息推广自己网站的方法:
1.网上许多站都喜欢做反向链接，所以，要从这里下手。
2.反向链接一般就是那么几个网站提供的，比如说，博搜。而这些网站为多家站点提借供反向链接时，一般是用的js调用，代码基本一样。所以用goole很容易找出这些网站。
3.现在你可以通过发送HTTP请求，在请求里加上REFERER字段来造假了。比如:发送这样一个HTTP请求:
GET / HTTP/1.1
HOST WWW.***.com
User-Agent:Mozilla/4.0 (compatible; MSIE 6.0; Windows NT 5.2; )
Accept:*/*
REFERER:http://www.ansys.org.cn/bbs
Connection:close
这样，对方接受到这个信息之后，他的”站外链接”就会显示www.ansys.org.cn/bbs,呵呵.
要实现大批量，要用好VB和java.前者　能够轻松的调用IE,java则有良好的多线程和同网络应用支持.做了多线程你可以一个小时发送数万条这样的信息。
当然，做好网站的关键是在你的网站要有内容。所以我自己是不会用这样的工具,贴出来给所以人，让别有用心的人知道，也要让无辜的站长知道。因为这样的工具只关注你的服务器是否记录了他的rererer信息，所以也根本不可能等着你的web 脚本调用完毕就关毕网络连接了.要解决，只能从服务器着手，直接封掉这样HTTP请求，其他的我还不知道什么解决方法。反过来看，不把显示这样的referer显示在你的外部链接列表中，但是这样要对方看过你的这个列表，知道你滤掉了这样的垃圾信息才不会发请求给你的服务器，否则虽然他想要在你的站上显示他的URL的目的没有达到，但你的服务器浪费了宝贵的资源去处理他的HTTP请求。
