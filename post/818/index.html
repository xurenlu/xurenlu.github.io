前端工程师在IE下调试页面之推荐装备

2009-01-14

Firefox下有Web developer,firebug安装也极方便，就不说了。 1:script editor : 需要修改IE选项－－>高级－－>禁止脚本调试(Internet Explorer)(去掉对钩) 2. debugbar: 这个有点儿FireBug的意思，但是差不少。 3. IE developer toolbar 这个也有点儿FireBug的意思，跟firebug差不少。 4. companion.JS IE下的js调试工具。 5. FireBug Lite FireBug 给IE开发的东西，可以协助调一下js. http://getfirebug.com/lite.html 5 安装IE6/IE7并存环境. 如果您已经装了IE 7:那您需要装MultipleIE. 它提供了各主流版本的 IE，虽然和原汁原味的版本有所差别，但对于我们作测试还是足够了。立刻下载 MultipleIE。（这种 IE 不支持条件注释符） 如果您只有IE6,还没有安装 IE7，可以下载一个第三方的 IE7，它不会破坏您原来版本的 IE，当然在使用方面和原汁原味的 IE7 也是有所区别的.请google之:IE7 Standalone。 至于使用方法，请询问google.
